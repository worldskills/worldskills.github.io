Events WorldSkills API

  • Description

Responsibilities

This API is responsible for storing WorldSkills events. This includes competitions, GAs, Champions Forums and regional events. It also stores all Competition related information like skills and sponsors.

Key Concepts

Each event has a “type” assigned to it. Events with the type “competition” also have a list of skills and a competition start and end date.
